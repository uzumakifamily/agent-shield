Three lines. Wrap your agent's action call with shield.run() — it intercepts the action, scores it, applies your policy rules, and either lets it through, pauses for your approval, or blocks it instantly. No architecture changes. Works with any Node.js agent, LangChain, AutoGen, or plain REST via curl. Full SDK and curl examples are in the Quick Test section.

Genuinely free — MIT licensed open source core. You run it on your own server, agent actions stay on your infrastructure, and there is no usage cap, rate limit, or expiry. The paid cloud tier adds the hosted web dashboard, Telegram approvals, managed storage, and 90-day audit retention — for teams who don't want to operate their own server.

No crash. shield.run() returns a result object with verdict: "BLOCK" or verdict: "PAUSE". Your agent receives this cleanly and can handle it gracefully — log it, notify the user, or retry later. For paused actions, Shield holds the action and fires a Telegram alert; once you approve or deny from the dashboard, a webhook notifies your agent. Your code stays in full control of what happens next.

On self-host: everything stays on your machine in a local SQLite database. Nothing is sent anywhere. On the cloud Pro tier: action metadata (type, risk score, verdict, timestamp) is stored in our managed database. Payloads containing PII are automatically redacted before storage — we never store raw email bodies, passwords, or personal data. You can export or delete your data at any time.

No Python SDK needed — use the REST API directly with requests. Add this check before any action your agent takes.

Helper function + file_delete example
import requests

SHIELD_API = "https://agentshield.allkinz.com"
API_KEY    = "YOUR_API_KEY"

def check_action(action_type, payload, agent="my-agent"):
    """Call AgentShield before any risky action."""
    r = requests.post(
        f"{SHIELD_API}/api/actions",
        headers={
            "Authorization": f"Bearer {API_KEY}",
            "Content-Type":  "application/json"
        },
        json={
            "action_type": action_type,
            "payload":     payload,
            "context":     {"agent": agent, "source": "api"}
        }
    )
    return r.json()

# --- Example: check before deleting a file ---
result = check_action("file_delete", {"path": "/data/users.csv"})

if result["verdict"] == "ALLOW":
    delete_file("/data/users.csv")        # safe to run
elif result["verdict"] == "PAUSE":
    print("Waiting for approval:", result.get("approval_id"))
elif result["verdict"] == "BLOCK":
    print("Action blocked:", result["reason"])
